# GlowCat 光芒小猫

GlowCat 是一个情感化服药提醒原型。它把服药动作重新定义为“喂食与陪伴”：用户按时打勾，猫咪自己吃掉灵力团子，并用光效、粒子和情绪反馈回应用户。

## 产品定位

一款以漫画小猫 IP 为核心的治愈型服药提醒工具。它不是单纯催促用户完成任务，而是建立一个轻量的共生循环：

- 用户通过按时服药照顾 GlowCat。
- GlowCat 通过视觉反馈、猫爪印章、粒子光效和情绪话术治愈用户。

## 核心功能

- 今日喂食清单：展示药品名称、提醒时间、剂量和完成状态。
- 自动喂食交互：用户点击打勾后，发光团子会自动飞向猫咪嘴边并触发进食反馈。
- GlowCat 状态机：支持等待喂食、傲娇催促、进食反馈、满足陪伴等状态。
- 动态灵力粒子：猫咪周围持续漂浮光点，喂食后触发粒子爆发与爱心/樱花飘散。
- 猫爪印章反馈：完成服药后，在对应任务卡上盖出猫爪印章。
- 手绘漫画日记：通过底部半屏面板添加或删除服药计划。
- 每日重置：以本地时间每日 04:00 为新一天起点，兼容熬夜场景。
- 昼夜氛围：白天使用暖色纸感背景，夜间切换为蓝绿色荧光氛围。

## 本地运行

这是一个零依赖静态原型，直接打开 `index.html` 即可。

也可以在项目目录启动本地静态服务：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

## 技术说明

- 使用原生 HTML/CSS/JavaScript 实现。
- 使用 `localStorage` 存储服药计划和当天打卡状态。
- 使用 Canvas 渲染猫咪周围的动态灵力粒子。
- 使用 CSS 绘制原创漫画招财猫风格角色，避免直接复刻既有版权角色。
- 使用 Web Vibration API 在支持的设备上提供轻量触觉反馈。

## 当前状态
